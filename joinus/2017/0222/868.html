UI设计师


岗位要求：

  1. 1.    精通企业级 UI 设计。能够实现复杂层级的信息展现，高效率的操作体验；

  2. 2.    精通 Photoshop 、 illustrator 等设计软件；

  3. 3.    能够进行动效设计者优先。能够设计控件在交互过程中的一些精细动画；

  4. 4.    能够制作字体图标（ font icon ）者优先；

  5. 5.    有跟前端工程师密切合作的经验。快速响应前端需求。 了解 JavaScript 、 CSS 或其它 HTML5 技术的优先；

  6. 6.    具有创业的热情和改造传统 2B 软件界面的动力；

  7. 7.    面试请提供个人近期作品；
